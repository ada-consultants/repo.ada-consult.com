APT Repository

Small APT repository to make our life a little bit easier

How-to

eval `cat /etc/lsb-release`
wget -O - http://repo.ada-consult.com/repo.ada-consult.com.gpg.key | sudo apt-key add - 
sudo sh -c "echo 'deb http://repo.ada-consult.com/ $DISTRIB_CODENAME main' > /etc/apt/sources.list.d/ada-consult.list"
